# OperaDesk | Gestão Estratégica de TI para Pequenas Empresas

O **OperaDesk** é uma solução de Service Desk desenvolvida especificamente para suprir a lacuna de gestão tecnológica em pequenas empresas. O projeto foca em eliminar processos manuais e planilhas de controle, oferecendo uma interface profissional e centralizada para incidentes e ativos.

---

## Proposta de Valor: Facilitar a Gestão de TI
Pequenas empresas muitas vezes operam no "caos controlado" de planilhas. O OperaDesk ataca essa dor através de:
* **Baixa Barreira de Entrada:** Interface intuitiva que dispensa treinamentos complexos.
* **Custo Operacional Reduzido:** Estrutura leve e funcional que organiza o fluxo de suporte sem exigir grandes investimentos em software.
* **Controle de Patrimônio:** Módulo de inventário focado em ativos críticos (notebooks, desktops e periféricos).

## Stack Técnica
* **Front-end:** HTML5, CSS3 e JavaScript.
* **Persistência de Dados:** Web Storage API (LocalStorage).
* **Arquitetura:** Focada em performance e fácil manutenção (Clean Code).

##  Funcionalidades Implementadas
* **Gestão de Incidentes em Pequena Escala:** Abertura, triagem e acompanhamento de chamados.
* **Dashboard de Visibilidade:** Indicadores simples e diretos para o monitoramento da operação.
* **Inventário de Hardware:** Cadastro e monitoramento de ativos corporativos.

## Roadmap de Expansão
O OperaDesk está sendo preparado para se tornar uma solução SaaS (Software as a Service) completa:
- [ ] Backend com autenticação segura e múltiplos inquilinos.
- [ ] Integração com Slack/WhatsApp para notificações de novos chamados.
